במהלך הקלטה: בצע פעולות בלייטרום (לחץ כפתורים, השתמש במקלדת). הקלטה לוכדת גם לחיצות עכבר.
הגדרה ראשונה
1
הרשאות נדרשות:
System Settings → Privacy & Security → Accessibility → הוסף LR Controller
System Settings → Privacy & Security → Input Monitoring → הוסף LR Controller

2
מצב גלובלי (מומלץ לXP-Pen):
לחץ "מצב גלובלי" — האפליקציה יוצרת CGEventTap אקטיבי שלוכד את כל המקשים.
מקשים ממופים נלכדים ומנועים מלהגיע ללייטרום — אין קונפליקט.
מקשים בלתי-ממופים עוברים לאפליקציה הפעילה כרגיל.
יש לאשר Accessibility בהגדרות המערכת.
3
מצב מכשיר (חלופה):
לחץ "↻ רענן" → בחר XP-Pen מהרשימה → לחץ עליו.
האפליקציה תתפוס את המכשיר בלעדית: רק כפתורי XP-Pen יפעילו מיפויים.
4
מיפוי פעולות:
טאב "פעולות" → לחץ למד → לחץ כפתור על XP-Pen.
5
מקרו עם לחיצות עכבר (למשל ישור אוטומטי):
טאב "מקרו" → + מקרו → ⏺ הקלט → בצע פעולות בלייטרום (כולל לחיצה על "Auto" בישור) → ⏹ עצור → הקצה מקש XP-Pen.
6
אם מופיעה שגיאה "damaged" או lr-helper:
הרץ בטרמינל: xattr -cr /Applications/LR\ Controller.app
